Codes defined by Care Connect Condition Relationship

System: https://fhir.nhs.uk/STU3/CodeSystem/CareConnect-ConditionRelationship-1

Code Description Definition
combined Combined This condition is combined/merged with the target condition to form a single logical condition.
grouped Grouped This condition is part of a group that includes the target condition as a member of the group.
replaces Replaces This condition replaces a previous condition (i.e. a revised diagnosis). The target condition is now obsolete.
evolved-from Evolved From This condition has evolved from the target condition.
onset Onset This condition follows-on from the target onset condition. Used when specifing a relationship to a previous onset condition when this condition episodicity is Review, Flare or End.
