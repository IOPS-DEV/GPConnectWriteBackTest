Codes defined by EMS Event Message Response Code

System: https://fhir.nhs.uk/STU3/CodeSystem/EMS-EventMessageResponseCode-1

Code Description Definition
R00 Message success Message Success
R01 Invalid NHS number format The NHS number provided in the message is not valid (based on being 10 digits long and the 10th digit being a check digit)
R02 NHS Number Does Not Exist on PDS The NHS number provided in the message does not match on PDS
R03 NHS Number Does Not Match Name And/Or/Gender And/Or DOB The NHS number provided in the message does not match the name and/or gender and/or DOB on PDS
R04 Service Not Entitled to Publish Event Type XXX The event type published is not allowed to be published by service YYY
R05 Supplier not accredited to publish messages The supplier system publishing the message has not been through the accreditation process
R06 Provider not accredited to publish messages The provider publishing the message has not been through the accreditation process
R07 SNOMED code does not exist The SNOMED code supplied in the FHIR message does
R08 Incorrect message format Message is not in XML or JSON format
R09 Incorrect Event The received event is not a recognised event
R10 Event does not conform The received event does not conform to the associated event type scheme
