CodeSystem definition: Person Relationship Type

Name: Person Relationship Type URL: https://fhir.nhs.uk/STU3/PersonRelationshipType-1
Version: 0.0.1 Status: Draft Purpose: This code system is defined to show a a relationship between one person and another.
Description: A code system that enumerates a relationship between one person and another. These relationships are not necessarily mutually exclusive, for example a wife may be her husband's next of kin.
